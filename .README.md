# API Test Framework (Postman + Newman)
End-to-end API test suite designed using Postman and executed automatically in a CI/CD pipeline with Newman (GitHub Actions).

## Scope
This project validates core REST API functionality, response structure, and data integrity through a realistic test workflow.

## Test flow
The collection covers a complete scenario:
* **GET resources** – retrieves data and validates the response.
* **POST entity creation** – creates a new resource using dynamic data.
* **DELETE cleanup** – removes the created resource to ensure repeatable and stable tests.

## Negative testing
In addition to the happy path, the suite includes negative test cases to verify correct API behaviour when invalid or incomplete data is sent. This ensures proper error handling, expected status codes, and system robustness.

## Key concepts
* REST API testing
* Dynamic data and request chaining
* Response validation
* Error handling and negative testing
* CI/CD integration with Newman

## Run locally
```bash
npm install -g newman
newman run postman/api_test_framework.collection.json -e postman/api_test_framework.environment.json --env-var "API_KEY=YOUR_KEY"
```
